# High Percent Visual Mod

## Funkcja
Mod sprawia, że w Minecraft 1.19.2 Fabric wszystkie tooltipy pokazujące `Dodatkowe obrażenia X%` będą pokazywały `Dodatkowe obrażenia 41%` cały czas, tylko wizualnie, nie zmieniając danych na serwerze.

## Budowa
1. Otwórz w IntelliJ IDEA / Eclipse.
2. Uruchom `gradlew build`.
3. Plik `.jar` znajdziesz w `build/libs`.
4. Wgraj `.jar` do folderu `mods`.
